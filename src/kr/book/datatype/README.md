# 자료형(Data Type)
### Java Data Type (자바 자료형)
<pre>
  <code> 
     Primitive Type							: 기본형
              ㄴ Boolean Type(boolean)					: 논리형 
              ㄴ Numeric Type						: 숫자형
                    ㄴ Integral Type 					: 적분형
                          ㄴ Integer Type(short, int, long) 		: 정수형
                          ㄴ Floating Point Type(float, double) 	: 실수형
                    ㄴ Character Type(char) 				: 문자형
     Reference Type 							: 참조형
              ㄴ Class Type						: 객체형
              ㄴ Interface Type						: 결합형
              ㄴ Array Type						: 배열형
              ㄴ Enum Type						: 열거형
              ㄴ etc.							: 기타
  </code>
</pre>
