# 상속(Inheritance)
<pre>
  <code>
	상속이란 객체지향 언어에서 다형성(Polymorphism)을 구현 하기 위한 방법이다.
	이미 구성된 클래스의 기능을 그대로 사용 하거나 부분적으로 재구성 하여 재사용성을 극대화시킨다.
	부모 클래스는 추상(abstract) 메소드를 만들고 자식 클래스에서 구체적인 코드를 만들 수 있다.
  </code>
</pre>
