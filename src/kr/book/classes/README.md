# 클래스(Class)
<pre>
  <code>
	하나의 객체(Object)를 만들기 위한 변수와 메소드의 집합이다.
	클래스는 일종의 어떤 형태를 만들기 위한 설계도와 같다.
	new 키워드를 이용하여 생성된 인스턴스 주소값을 변수 값으로 참조한다.
	생성된 인스턴스 주소값은  Heap area(힙 영역)의 주소값과 같다.
  </code>
</pre>
