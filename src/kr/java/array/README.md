# 배열(Array)
<pre>
  <code>
	배열은 연관된 데이터를 모아서 관리하기 위하여 사용하는 데이터 유형이다.
	변수와 달리 여러 개의 데이터를 저장하고 필요할때 찾아서 사용이 편리하다.
	종류 : 기본형 배열, 참조형 배열
  </code>
</pre>
