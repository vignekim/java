# 패키지(Package)
<pre>
  <code>
	자바(Java) 코드를 하나의 프로젝트(그룹)으로 만들기 위한 구조 경로를 말한다.
	클래스 생성 시 자바 파일을 생성하기 때문에 일반 파일과 동일하게 관리 할 수 있다.
	패키지 구성은 폴더와 폴더 관계를 나타내기 위하여 '.'을 사용하여 구분한다.
  </code>
</pre>
